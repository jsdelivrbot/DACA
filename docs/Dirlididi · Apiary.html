                      Introduction

                      API do Dirlididi - Um Sistema para submissão online de exercícios

                      Reference

                      Usuários

                      Attributes

                      Adicionar um novo usuário

                      Adiciona um novo usuário ao sistema. Recebe um JSON com email e senha do usuário e responde com um token de acesso.

                      Autenticação

                      Attributes

                      Autenticar usuário

                      Autentica um novo usuário ao sistema. Recebe um JSON com email e senha do usuário e retorna um token de acesso.

                      Problemas

                      Attributes

                      Listar todos os problemas

                      Retorna todos os problemas disponíveis de acordo com um critério de busca.

                      Criar um novo problema

                      Cria um novo problema (inicialmente não publicado). Requer um token de acesso.

                      Problema

                      Attributes

                      Detalhar um problema específico

                      Retorna informações sobre um problema específico.

                      Atualizar um problema específico

                      Atualiza um problema já criado, se o token fornecido corresponder ao criador do problema ou a um administrador do sistema.

                      Remover um problema específico

                      Remove um problema da lista de problemas disponíveis. É preciso ser o criador do problema ou administrador para realizar essa ação.

                      Testes

                      Attributes

                      Detalhar os testes de um problema específico

                      Retorna todos os testes relativos a um problema.

                      Atualizar os testes de um problema específico

                      Substitui todos os testes de um problema especifico por outro conjunto de testes. É preciso ser o criador do problema ou administrador para realizar essa ação.

                      Solução

                      Attributes

                      Detalhar uma solução de um problema

                      Retorna informações sobre a última solução enviada para um problema.

                      Submeter um solução para um problema

                      Envia uma nova solução para um problema. Deve conter uma lista de strings contendo a saída para cada um dos testes do problema. Retorna se o problema foi ou não resolvido.

                      Estatísticas gerais

                      Attributes

                      Listar estatísticas gerais

                      Retorna o número de problemas publicados e o número de pessoas que submeteram alguma solução.

                      Estatísticas do usuário

                      Attributes

                      Listar estatísticas do usuário

                      Retorna o número de problemas resolvidos pelo usuário

                      Ranking de usuários

                      Attributes

                      Listar ranking de usuários

                      Retorna a lista de usuários ordenada pelo número de questões resolvidas
